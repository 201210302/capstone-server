# 가로쓰레기통 알리미 (Server)
- 가로쓰레기통 알리미 앱의 서버 레포입니다.

## 사용기술
- 언어 : Javascript, Typescript
- 프레임워크 : NestJS

## 주요 패키지
- Swagger : API문서 작성을 위한 패키지입니다.
- TypeOrm : DB와 연동하기 위한 ORM패키지입니다.
